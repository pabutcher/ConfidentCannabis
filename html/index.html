Wholesale price of Cannabis flower


This API only contains one function "generate_wholesale_price" which returns the estimated pretax wholesale price per pound for a sample of cannabis

Wholesale price estimation

Request: http://localhost:8080/generate_wholesale_price

Only Post Requests Accepted
The output is a JSON-file in the following format:

{
  "success": true/false,
  "payload": {
    "pplb_pretax":estprice
  }
}
Where "estprice" is the estimated wholesale price per pound
If an error is present it is returned as a string in "pplb_pretax"

Expected input

A JSON payload is neccessary as input. All fields must be present and of correct type or an error is returned.
